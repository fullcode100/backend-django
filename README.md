# Route and Formatting
- /api/team/
```JSON
[
	{
		"id"
		"name"
		"win"
		"lose"
		"draw"
		"goal_masuk"
        "goal_kebobolan"
        "points"
        "selisih_goal"
        "banyak_match"
	},
	....
]
```
- /api/team/{pk}/
```JSON
{
		"id"
		"name"
		"win"
		"lose"
		"draw"
		"goal"
		"team_players": [
			{
				"id"
				"name"
				"profile_picture"
			},
			....
		]
}
```
- /api/team/{pk}/players/
```JSON
{
	"id"
	"name"
	"team_players": [
		{
			"id"
			"name"
			"profile_picture"
		}
	]
	
}
```
